1. 스파크 사이트에서 원하는 버전을 미러 버전으로 다운로드(root권한)

2. tar -xf로 압축해제
```bash
홈디렉토리$ tar -xf spark-3.0.2-bin-hadoop2.7
```

3. 스파크 디렉토리 안으로 들어가기
```bash
cd spark-3.0.2-bin-hadoop2.7
```

4. 실행
```bash
홈디렉토리/spark-3.0.2-bin-hadoop2.7$ bin/spark-shell
```

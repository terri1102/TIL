1. cat : concatenate (여러) 파일명을 받아서 화면에 뿌려줌
```linux
$ cat [옵션] [파일명]
$ cat /etc/os-release
```

2. free 메모리 상태 확인

```linux
$ free -h
```

3. uname : 설치된 커널 버전 확인

```linux
$ uname -r
```

4. gui로 로그인하고 싶을 떄
```linux
$ sudo systemctl isolate graphical.target
```

5. 텍스트 모드로 로그인(메모리 사용량 줄임)

```linux
sudo systemctl isolate multi-user.target
```

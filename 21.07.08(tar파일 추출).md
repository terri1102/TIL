파일 이름으로 된 폴더 생성 후 그 아래에 압축이 풀어짐
```python
import tarfile
fname = "empatheticdialogues.tar.gz"
if fname.endswith("tar.gz"):
    tar = tarfile.open(fname, "r:gz")
    tar.extractall()
    tar.close()
elif fname.endswith("tar"):
    tar = tarfile.open(fname, "r:")
    tar.extractall()
    tar.close()
```

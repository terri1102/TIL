python 2.x 문법. 3.x버전에서도 사용 가능. 

```python
super(ClassName, self).__init__()
super(ClassName, self).__init__(**kwargs)
```

python 3.x 문법
```python
super().__init__()
super().__init__(**kwargs)
```

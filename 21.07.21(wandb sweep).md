1. 로그인

```python
import wandb

# 1. Start a W&B run
wandb.init(project='project name', entity='wandb ID')
```
2. sweep으로 찾을 하이퍼 파라미터 정의
```python
import wandb

sweep_config = {
  "name" : "my-sweep",
  "method" : "random",
  #"optimizer" : ["adam","adamw"],
  "parameters" : {
    "epochs" : {
      "values" : [16, 32],
    },
    "dropout" : {
        "values" : [0.1, 0.3]
    } 
  }
}

sweep_id = wandb.sweep(sweep_config)
```
3. train 실행 코드 아래에 적기
```python
wandb.agent(sweep_id, function=train, count=5)
```
